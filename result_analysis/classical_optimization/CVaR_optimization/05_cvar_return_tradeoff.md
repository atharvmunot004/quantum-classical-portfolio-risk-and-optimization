# CVaR-Return Tradeoff

## CVaR Return Frontier

The CVaR-return frontier illustrates the efficient tradeoff between expected return and tail risk (CVaR). Portfolios on the frontier maximize return for a given CVaR level, demonstrating the effectiveness of CVaR optimization in managing downside risk.
